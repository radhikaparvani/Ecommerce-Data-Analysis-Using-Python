# Ecommerce Data Analysis Using Python

## Project Overview

This project is an end-to-end ecommerce data analysis project created using Python. The main objective of this project is to analyze customer behavior, sales trends, product performance, customer segmentation, retention, anomaly detection, and business insights using real-world style ecommerce datasets.

## Tools and Libraries Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset Used

The project uses four ecommerce datasets:

- Customers dataset
- Orders dataset
- Order items dataset
- Products dataset

These datasets were merged to create a final master dataset for complete analysis.

## Key Insights

- Sales trends showed clear patterns across daily, weekly, and monthly levels.
- A small group of high-value customers contributed significantly to total revenue.
- Repeat customers were more valuable than one-time buyers.
- Some product categories performed strongly, while others showed lower revenue contribution.
- Customer segmentation helped identify Premium, Regular, and Low Value customers.
- Retention analysis helped classify active and inactive customers.
- Anomaly and fraud detection logic helped identify suspicious transactions.

## Business Recommendations

- Focus on high-value and repeat customers through loyalty programs.
- Improve underperforming product categories with discounts or promotions.
- Use targeted offers for inactive or high-risk customers.
- Monitor suspicious transactions using rule-based fraud detection.
- Track sales trends regularly to improve business planning.
